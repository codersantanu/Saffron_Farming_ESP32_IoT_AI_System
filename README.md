# 🌸 SaffronSense AI

### Smart Farming using IoT, Machine Learning & Cloud ☁️🤖

---

## 📌 Overview

**SaffronSense AI** is an intelligent IoT-based smart farming system designed specifically for saffron cultivation. It integrates real-time sensor monitoring, cloud computing, and machine learning to optimize crop conditions, improve yield, and enable data-driven agricultural decisions.

---

## 🚀 Key Features

* 🌡️ Real-time environmental monitoring (Temperature, Humidity, Gas, Soil Moisture)
* 📡 IoT-enabled data collection using ESP32
* ☁️ Cloud integration with Firebase for secure data storage
* 🤖 Machine Learning models for prediction & automation
* 📱 Mobile app for remote monitoring & control
* 🌐 Web dashboard for live visualization and analytics

---

## 🧠 System Architecture

```text
Sensors → ESP32 → Cloud (Firebase) → ML Model → Web/Mobile App
```

---

## 🛠️ Technology Stack

### 🔌 Hardware

* ESP32 Microcontroller
* MQ135 Gas Sensor
* DHT22 Temperature & Humidity Sensor
* Soil Moisture Sensor

### 💻 Software & Tools

* Arduino IDE
* Firebase (Realtime Database / Firestore)
* HTML, CSS, JavaScript (Web Dashboard)
* Python (Machine Learning Models)
