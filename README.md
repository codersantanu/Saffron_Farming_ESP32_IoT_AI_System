# 🌸 SaffronSense AI

### Smart Farming using IoT, Machine Learning & Cloud ☁️🤖

---

## 📌 Overview

**SaffronSense AI** is an intelligent IoT-based smart farming system designed specifically for **saffron cultivation**.
It combines **real-time sensor monitoring, cloud computing, and machine learning** to optimize environmental conditions, improve crop yield, and enable **data-driven agricultural decisions**.

---

## 🚀 Key Features

* 🌡️ Real-time environmental monitoring
  *(Temperature, Humidity, Gas, Soil Moisture)*
* 📡 IoT-based data collection using **ESP32**
* ☁️ Cloud integration with **Firebase** for secure storage
* 🤖 Machine Learning models for prediction & automation
* 📱 Mobile app for remote monitoring & control
* 🌐 Web dashboard for live visualization & analytics

---

## 🧠 System Architecture

```
Sensors → ESP32 → Firebase Cloud → ML Model → Web/Mobile App
```

---

## 🛠️ Technology Stack

### 🔌 Hardware

* ESP32 Microcontroller
* MQ135 Gas Sensor
* DHT22 Temperature & Humidity Sensor
* Soil Moisture Sensor

### 💻 Software & Tools

* Arduino IDE
* Supabase
* HTML, CSS, JavaScript (Web Dashboard)
* Python (Machine Learning Models)

3. Configure Firebase credentials
4. Run Web Dashboard
